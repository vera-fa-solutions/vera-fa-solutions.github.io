                                                                                                                                                            Transaction types

                                                                                                                                                            In the FA Platform, each security type has a set of transaction types you can use. For a list of transaction types available by default, see Transaction types for security transactions and Transaction types for cash transactions.Transaction types for security transactionsTransaction types for cash transactions

                                                                                                                                                            Modifying transaction types is not recommended. Transaction types are defined to work in specific scenarios and changes can cause, for example, incorrect performance calculations. Instead of editing the transaction types, you can rename them or add new ones.

                                                                                                                                                            If you want to modify the name of an existing transaction type, you can use the Translate button in Transaction types (Preference, Transactions section). You should not modify the transaction type name directly.

                                                                                                                                                            Add a transaction type

                                                                                                                                                            You can add a new transaction type if the standard types do not include a suitable type for your needs. Use a similar existing type as an example. If you need to create a transaction type from scratch, contact FA support to ensure that no problems occur, for example, with profit calculations.

                                                                                                                                                            To add a new transaction type, do the following:

                                                                                                                                                            1. Open Transaction types (Preference, Transactions section) and click Add transaction type.

                                                                                                                                                            2. Fill in the fields and make sure all effects are correct (see Preference - Transactions in FA Back reference).Preference - Transactions

                                                                                                                                                            3. Click Save.

                                                                                                                                                            Questions about transaction types

                                                                                                                                                            I get several different cost and profit transactions from my custody. I want to record all of them in FA. Can I add a transaction type for this?

                                                                                                                                                            Yes, you can add a new transaction type for this. Copy the transaction type effects from a similar standard transaction type and link the new transaction type to the necessary security types.
