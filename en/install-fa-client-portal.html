                                                                                                                                                                              Install FA Client Portal

                                                                                                                                                                              You can install FA Client Portal from Optional packages. After the installation, set up the user access:

                                                                                                                                                                              1. Provide users with access to the app. To do this, assign the correct user role to the users who are supposed to access FA Client Portal. If you don't have such a role available in FA Admin Console, User management view, ask FA to create one for you.

                                                                                                                                                                              2. Provide users with access to data. For each user, add a linked contact whose portfolios they are allowed to see.

                                                                                                                                                                              You can access FA Client Portal at https://<instance>.clientportal.fasolutions.com.
