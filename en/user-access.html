                                      User access

                                      Follow the instructions below to set up access to FA Client Portal:

                                      • Access for end users - Create a user and specify contacts and portfolios the user can access.

                                      • Access for troubleshooting - Set up view-only access to client information for advisors, support service, and so on.

                                      Access for end users

                                      FA Client Portal provides end clients with access to investment information. The user can access:

                                      • Their own portfolios.

                                      • Portfolios of the contacts the user represents – for example, if the user represents their child or a company they work for. A representative can switch between the contacts in FA Client Portal by clicking .

                                      To provide access to FA Client Portal, you need to:

                                      1. Add a new user as described in Manage users in FA Admin guide. Assign the user CLIENT PORTAL role.Manage users

                                      2. Set up the user’s access to client information stored in FA Back:

                                        • To let the user access their own investment information, make sure the user's contact is specified as a primary contact for the portfolio (see Contacts) and link this contact to the user in FA Admin console (to do this, follow the steps in Link a contact to the user).Link a contact to the user

                                        • To let a representative access the linked contacts’ information in FA Client Portal, add the user as the contact’s representative in the Contact window, Representatives tab.

                                      Access for troubleshooting

                                      View-only mode lets advisors and support service specialists access the same view as the client sees, without being able to make any modifications. To access client accounts in FA Client Portal view-only:

                                      1. Assign your user "CLIENT PORTAL IMPERSONATE" role in FA Admin Console. See Assign user roles for detailed instructions.Assign user roles

                                      2. View FA Client Portal for a contact in the impersonate mode using one of the methods:

                                        • Open your browser and enter the link https://<instance>.fasolutions.com/public/myinvestments/impersonate/<client's contact ID>/.

                                        • In FA Back, right-click the contact in the Overview and choose "Impersonate in Client Portal".
