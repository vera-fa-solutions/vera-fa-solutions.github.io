                                        FA audit logs

                                        FA audit logs let you track user activities related to sensitive customer data and changes in preferences that can modify the business logic.

                                        Customer information logs

                                        FA records logging in and out, opening windows, making selections on Overview, searches, report generations, PDF and XLS exports. Each audit trail entry includes details about the user, time, and screen, and information the user accessed or exported.

                                        Changes in system settings

                                        FA tracks changes in system settings that can affect FA functionality and system logic.

                                        Action

                                        Message

                                        Attachment

                                        Change preferences

                                        Create, modify or delete Account set in Preference → Bookkeeping → Account set

                                        User [username] created/modified/deleted account set [account set code]

                                        Changes in JSON format. In case of creating or deleting an account set, all account set attributes and accounts are listed. Modification shows the change, the old values, and the new ones.

                                        Modify Analytics+ columns in Preference → Content and translations → Analytics+ columns

                                        User [username] changed Analytics+ custom column definition for [custom column code]

                                        Changes in JSON format. In case of creating or deleting a column, all column attributes are listed. Modification shows the change, the old values, and the new ones.

                                        Modify permissions in Preference → Administration → Permissions

                                        User [username] changed permissions: [permission change]

                                        Changes in JSON format: old permission set and the new one.

                                        Create, modify or delete a posting rule in Preferences → Bookkeeping → Posting rules

                                        User [username] created/modified/deleted posting rule in rule set [ruleset code] id [rule ID]

                                        Changes in JSON format. In case of creating or deleting a posting rule, all posting rule attributes are listed. Modification shows the change, the old values, and the new ones.

                                        Import a posting rule import in Preferences → Bookkeeping → Posting rules

                                        User [username] imported posting rules in rule set [rule set code]

                                        Changes in JSON format: all attributes of all rules in the imported file as one list.

                                        Create, modify or delete security type changes in Preference → Securities → Security types

                                        User [username] created/modified/deleted security type [security type code]

                                        Changes in JSON format. In case of creating or deleting a security type, all security type attributes are listed. Modification shows the change, the old values, and the new ones.

                                        Change API or processes configurations

                                        Generate or delete API token in Administration → API → Key management

                                        User [username] created/deleted API token [token name]

                                        Change process configuration in Administration → Processes

                                        User [username] added/modified/deleted configuration of process [processId]

                                        Changes in JSON format. In case of creating or deleting a configuration, all configuration attributes are listed. Modification shows the change, the old values, and the new ones.
