                                                                                                              FA Advisor user accesses

                                                                                                              To set up user access to FA Advisor and give a user access to customer data, do the following:

                                                                                                              1. Give a user access to the FA Advisor app by assigning them the correct user role in FA Admin Console, User management view. If you do not have such a role available in the User management view, ask FA to create one for you.

                                                                                                              2. In the FA Admin Console, User management view, link the user to a contact (see Link a contact to the user).Link a contact to the user

                                                                                                              3. In FA Back, add the contact you linked to in step two as a representative for the contacts they represent in the Contact window, Representatives tab (see Contact window).Contact window

                                                                                                              Set up access to customer investments in Client Portal

                                                                                                              A user can access a view-only version of a customer's FA Client Portal in the Customers view. To give a user access, assign the correct role to them. See Assign user roles for detailed instructions.Assign user roles
