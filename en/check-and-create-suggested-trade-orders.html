                                                                                        Check and create suggested trade orders

                                                                                        Once you are satisfied with the target values you see, you can proceed to trade order preview.

                                                                                        1. If you want to ignore the cash balance and buy with negative cash if needed, select Ignore cash balance.

                                                                                        2. Click Preview to view suggested trade orders.

                                                                                          • Orders are created with the "Open" status.

                                                                                          • The system suggests using a portfolio account in the security currency. If such account is missing, the default portfolio account is used.

                                                                                        3. Select the ones you want to create and proceed with one of the following:

                                                                                          • Check limits to see if the trade orders are within the portfolio limits.

                                                                                          • Create selected orders. You can find the created orders in the Trade orders view.

                                                                                          • Create simulations.
