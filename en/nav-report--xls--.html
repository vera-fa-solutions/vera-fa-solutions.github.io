                        NAV report (XLSX)

                        NAV report contains details of NAV calculation that you can see in the NAV window tabs. You can download NAV report in the XLSX format in the NAV window (see NAV window for details).NAV window

                        The Preview summary section shows the summary of the current NAV calculation.

                        The Share classes section shows NAV calculation by share class (see the Table columns description in Share classes tab).

                        The Portfolio section shows portfolio positions (see the Table columns description in Portfolio tab). You can choose the portfolio position information that is included in the report. To do this, select the columns in the Portfolio tab by clicking the button before you generate the report.Portfolio tab

                        The Warnings section shows four types of warnings generated as a result of NAV calculation. For details, see the Warnings tab section.Warnings tab

                        The Limits section shows post-trade limits analysis generated as a result of NAV calculation. You need to set up the limits for your fund in FA Back to get this information displayed in NAV report. For details, see Limit definitions.Limit definitions
