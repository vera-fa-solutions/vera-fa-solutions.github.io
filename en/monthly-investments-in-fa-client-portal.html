                                    Monthly investments in FA Client Portal

                                    The user can set up monthly investments to buy securities regularly. The user specifies which securities to buy, the amount to spend, and defines the schedule. After that, FA automatically creates trade orders for the user based on the entered information.

                                    Enable monthly investments in FA Back

                                    To enable monthly savings functionality in FA Back:

                                    1. Follow the Set up monthly investments instructions in FA Admin guide to define the trade order generation details.Set up monthly investments

                                    2. To enable monthly investment functionality for the user, include the client's portfolios in the CP_MONTHLYINVESTMENTS portfolio group. The user can set up monthly investments only for portfolios included in the CP_MONTHLYINVESTMENTS portfolio group.

                                    Set up monthly investments in FA Client Portal

                                    To define the monthly investments in FA Client Portal, the user chooses Monthly investments in the top-right menu and fills in the following information:

                                    Portfolio section

                                    The Portfolio section lets the user choose the investment portfolio and the amount to invest in the portfolio currency. The portfolio selection shows only portfolios included in the CP_MONTHLYINVESTMENTS portfolio group.

                                    You can define a minimum monthly investment per portfolio by adding portfolio key figure CP_MI_MIN_AMOUNT in FA Back.

                                    Security section

                                    The Security section lets the user search and choose the securities to invest in. The list shows securities with the Tradeable tag. The user can search by security name or ISIN, and select the security from the results list below.

                                    Investment distribution section

                                    The Investment distribution section lets the user define the percentage or amount to invest in each security. The entered percentage or amount must take into account the minimum trade amount shown for each security (it is fetched from the security's Minimum trade amount fields in the Cost info tab).

                                    The Distribute evenly button divides the investment amount evenly between the chosen securities.

                                    Investment schedule section

                                    The Investment schedule section lets the user define the months and dates to generate trade orders for.

                                    Summary section

                                    The Summary section provides an overview of the monthly investments settings and lets the user confirm the choices they made.
