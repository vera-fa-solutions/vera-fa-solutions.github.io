                                      Manage portfolio credit

                                      Overview

                                      Portfolio credit is a feature that allows you to give loans to end users using their portfolio holdings as collateral. The credit is calculated based on a percentage of the market value of the portfolio. The following sections describe the logic behind portfolio credit calculations and guide you through giving portfolio credit to an end user.

                                      Portfolio holdings as collateral for a loan

                                      The FA Platform calculates the Collateral value of the portfolio, but the total amount of credit available to use is decided by you. The collateral value is based on all of the positions in the portfolio and can serve as guidance for your decision on the total amount of credit available to the end user.

                                      Portfolio credit calculation logic

                                      Each security in the portfolio must have a defined Collateral ratio for portfolio credit to be calculated. This is because it is needed to calculate the Collateral value of each security and the whole portfolio. In practice, the Collateral value of an individual security is determined by multiplying the following values:

                                      1. Market value of the security.

                                      2. Collateral ratio of the security. The Collateral ratio of a security is a percentage of its market value.

                                      For example, if security A has a collateral ratio of 40% and a market value of 100,000 euros, and security B has a collateral ratio of 10% and a market value of 50,000 euros, the Collateral value of the holdings is calculated as follows:

                                      100,000 * 0,40 + 50,000 * 0,10 = 45,000 euros

                                      The Collateral value is the summed-up value of all portfolio holdings that is shown in Portfolio window → Portfolio credit as guidance for the amount of total credit you want to give the portfolio. The collateral value is calculated and shown in the portfolio currency.

                                      Assign credit to a portfolio

                                      Set Collateral ratio for portfolio holdings

                                      In order for the Collateral value to be calculated for a portfolio, the positions within it must each have a defined Collateral ratio which is a percentage of the security's market value.

                                      You can set the Collateral ratio for a security in Security window → Key figures. Enter the value as a percentage without the percent sign. The Collateral ratio can also be imported along with other security information.

                                      For more details on key figures for securities and importing security information, see Security key figures and File formats for importing securities and security prices.Security key figures

                                      Manage credit in the Portfolio credit tab

                                      You can manage portfolio credit in Portfolio window → Portfolio credit. The fields in this tab are described in detail in .

                                      The Portfolio credit tab shows the Collateral value for the portfolio in its currency, calculated by the system as described above. Click Show calculation details to see how the value is calculated: The details show a table of the positions in the portfolio with columns that display their individual market values and collateral ratios.

                                      Enter the total amount of credit you want to assign to the portfolio in the Portfolio credit field. You can base this value on the Collateral value. After entering the value, click Save or Save & Close to save your changes. The system automatically calculates the amount of Used credit and Available credit based on the portfolio investments and the total amount of credit you assign to the portfolio.
