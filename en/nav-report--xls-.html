                          NAV report (XLS)

                          NAV report contains details of NAV calculation that you can see in the NAV window tabs. You can download NAV report in the XLS format in the NAV window (see NAV window for details).NAV window

                          The Preview summary section shows the summary of the current NAV calculation.

                          The Share class section shows NAV calculation by share class (see the Table columns description in Share classes tab).Share classes tab

                          The Portfolio section shows portfolio positions (see the Table columns description in Portfolio tab). You can choose the portfolio position information that is included in the report. To do this, select the columns in the Portfolio tab by clicking the button before you generate the report.Portfolio tab

                          The Warnings section shows four types of warnings generated as a result of NAV calculation (for details, see the Warnings tab section).Warnings tab
